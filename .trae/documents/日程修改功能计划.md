## 日程修改功能实现计划

### 修改目标
在应用中直接丝滑地修改日程，无需手动编辑 data.json 文件。

### 实现方案

#### 1. 后端 API (server.js)
- 添加 `POST /api/schedule` - 创建新日程
- 添加 `PUT /api/schedule/:date` - 修改指定日期的日程
- 添加 `DELETE /api/schedule/:date` - 删除指定日期的日程

#### 2. 前端界面 (index.html)
- 在日期详情弹窗中添加"编辑"按钮
- 添加编辑弹窗，支持修改：
  - 日期标题 (title)
  - 亮点/重点 (highlights)
  - 里程碑 (milestone)
  - 时间段列表 (timeSlots) - 可增删改每个时间段

#### 3. 交互流程
1. 点击日历中有日程的日期 → 打开详情弹窗
2. 点击弹窗中的"编辑"按钮 → 进入编辑模式
3. 修改完成后点击"保存" → 发送 PUT 请求到后端
4. 后端更新 data.json → 前端刷新显示

#### 4. 样式优化
- 编辑表单采用与现有科技风格一致的深色主题
- 添加按钮 hover 动画效果
- 确保编辑体验丝滑流畅
