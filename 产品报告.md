## 一、产品概述

### 1.1 产品定义

PlanMosaic（计划马赛克）是一款**智能学业规划与词汇学习综合平台**，将日程管理、词汇学习与AI智能助手深度融合，为学生和知识工作者提供一站式的时间管理与知识学习解决方案。

### 1.2 产品愿景

> 让每个人的时间都有迹可循，让每次学习都高效有趣。

通过AI技术与科学的规划方法论，帮助用户建立系统化的日程管理习惯，同时将碎片化时间转化为有效的词汇学习机会。

### 1.3 产品命名

- **PlanMosaic**：主品牌名，寓意将碎片化的时间安排拼合成有序的整体
- **WordMosaic**：词汇学习子品牌，寓意将零散的单词编织成完整的语言图景
- **Mosa**：AI助手名称，贯穿日程管理与词汇学习的智能中枢

## 二、目标用户

### 2.1 核心用户画像

| 用户类型 | 典型特征 | 核心需求 |
|---------|---------|---------|
| 大学生 | 课程多、DDL多、备考压力大 | 课程表管理、考试DDL追踪、四六级/考研词汇学习 |
| 研究生 | 科研任务重、时间自主性强 | 长期项目规划、文献阅读安排、专业英语词汇积累 |
| 医学生 | 课程体系复杂、专业词汇量大 | 课程表管理、医学英语词汇记忆、实习安排规划 |
| 考研备考者 | 学习计划严格、复习周期长 | 系统化复习安排、考研词汇、进度追踪 |
| 终身学习者 | 自主学习为主、时间碎片化 | 灵活的时间安排、个性化词汇学习、AI辅助学习 |

### 2.2 用户痛点分析

1. **日程管理痛点**
   - 课程、作业、考试等日程分散，难以统一管理
   - 手动排课繁琐，冲突检测依赖人工
   - 缺乏长期目标与短期安排的衔接机制

2. **词汇学习痛点**
   - 背单词枯燥，缺乏即时反馈
   - 词库固定，无法针对特定专业领域学习
   - 学习进度难以追踪，容易半途而废

3. **效率提升痛点**
   - 日程管理与学习割裂，无法联动
   - 缺乏智能化建议，规划效率低
   - 跨设备数据不同步，体验断裂

### 3.2 核心功能详述

#### 3.2.1 智能日程管理

**日历视图**
- 月历视图：以月为单位展示所有日程安排，有日程的日期高亮显示
- 周历视图：以周为单位展示每日时间段安排，直观展示时间分配
- 点击任意日期即可查看/编辑该日的详细安排

**时间段管理**
- 支持添加精确到分钟的时间段（如 09:00-10:30）
- 每个时间段包含：活动名称、详细说明、图标标识
- 支持批量操作：批量添加、批量修改、批量删除
- 支持周期性日程：每日/每周/工作日重复

**任务管理**
- 每日任务清单，支持预估时间和实际时间记录
- 任务完成追踪，可视化进度条
- 任务备注功能，记录关键信息
- 与时间段关联，形成完整的时间-任务闭环

**大型任务（DDL管理）**
- 支持创建带有截止日期的大型任务
- 区分短期任务和长期任务
- 支持将大型任务分解为子任务，分配到具体日期
- 自动追踪逾期状态（待完成/已完成/已逾期）
- 完成时记录完成时间，形成历史记录

**课程表系统**
- 一键创建学期课程表，设定日期范围和总周数
- 支持单双周课程安排
- 课程图标自动匹配（数学∫、英语Aa、物理F等）
- 课程调换、课程负载分析
- 课程表导入/导出功能

**日程模板**
- 将常用日程保存为模板
- 快速应用模板到目标日期
- 支持模板管理（创建/查看/删除）

**冲突检测**
- 自动检测时间段重叠
- 在添加日程时实时提示冲突
- AI助手可主动检测全日程表中的冲突

#### 3.2.2 词汇学习系统 (WordMosaic)

**词库体系**

| 词库 | 内容 | 词汇量 | 适用人群 |
|------|------|--------|---------|
| 大学英语 Book1 | 四级核心词汇 | 约3000词 | 四级备考 |
| 大学英语 Book2 | 六级核心词汇 | 约5000词 | 六级备考 |
| 大学英语 Book3 | 考研英语核心词汇 | 大量词汇 | 考研备考 |
| 医学学术英语 | 医学专业词汇 | 大量词汇 | 医学生 |
| 自定义词库 | AI生成/用户导入 | 不限 | 所有用户 |

**学习模式**
- **英译中模式**：显示英文单词+音标+释义，从4个中文选项中选择正确翻译
- **中译英模式**：显示中文释义，手动输入英文单词拼写
- 支持跳过不熟悉的单词
- 实时反馈正确/错误

**学习管理**
- 按字母/单元分组筛选
- 分组锁定功能（防止学习中途误切换）
- 随机打乱学习顺序（Fisher-Yates算法）
- 键盘快捷键支持（Enter提交/跳过）

**错题本系统**
- 自动记录答错和跳过的单词
- 支持导出错题为文本文件
- 一键进入错题复习模式
- 清空错题功能

**学习统计**
- 总词汇量、已学习、正确、错误数量
- 正确率百分比计算
- 各词库独立进度追踪
- 支持重置学习进度

**AI词汇生成**
- 按主题生成词汇列表（如医学、GRE、雅思、日常等）
- 支持上传文件（txt/md/json）提取词汇
- 流式实时生成，边生成边展示
- 生成的词汇可保存为自定义词库
- JSON格式自动提取和验证

#### 3.2.3 Mosa AI智能助手

**日程助手模式**

用户通过自然语言与Mosa交互，Mosa理解用户意图并自动执行日程操作。支持的操作包括：

| 操作类别 | 具体能力 |
|---------|---------|
| 查看日程 | "帮我看看明天有什么安排" / "这周有哪些课" |
| 添加日程 | "明天上午9点到11点安排复习高数" / "每周三下午2点上英语课" |
| 修改日程 | "把明天的复习时间改到下午" / "删除下周一的物理课" |
| 任务管理 | "添加一个任务：完成论文初稿，预计5小时" / "标记英语作业已完成" |
| 大型任务 | "创建DDL任务：期末考试，截止日期1月15日" / "分解这个大任务" |
| 课程管理 | "帮我创建这学期的课程表" / "调换周一和周二的课" |
| 冲突检测 | "检查这周有没有时间冲突" |
| 模板管理 | "把这个日程保存为模板" / "应用模板到下周一" |
| 分析优化 | "分析我这周的时间使用情况" / "帮我优化日程安排" |

**深度规划模式**

面向战略级人生规划，提供5大专业分析工具：

1. **价值量化分析** (value_monetization) - 量化活动的长期价值
2. **ROI计算器** (roi_calculator) - 计算时间投入的回报率
3. **里程碑规划** (milestone_planner) - 制定阶段性目标与里程碑
4. **SWOT分析** (swot_analysis) - 优势/劣势/机会/威胁四维分析
5. **决策矩阵** (decision_matrix) - 多维度量化决策支持

**AI技术特性**
- 支持多轮对话，保持上下文记忆（最近8轮）
- 用户画像自动构建，实现个性化建议
- 对话自动归档压缩（超过50条自动归档至5%体积）
- 操作建议机制：破坏性操作需用户确认（Proposal机制）
- 支持两个AI提供商：DeepSeek（深度求索）和Qwen（通义千问）

#### 3.2.4 用户系统

**账户管理**
- 用户名/密码注册与登录
- 自动登录（记住密码）
- 密码AES-256-GCM加密存储
- 多用户切换，数据完全隔离
- 安全登出

**数据同步**
- 基于Supabase云端同步
- 登录时自动拉取云端数据
- 本地数据合并策略（云端数据补充本地）
- 支持手动触发同步
- 离线可用，联网后同步

#### 3.2.5 通用功能

**主题系统**
- 亮色主题：温暖的纸质色调（#F9F8F6背景）
- 暗色主题：深色调（#1A1918背景）
- 毛玻璃效果、纸张纹理叠加
- CSS自定义属性实现主题切换

**数据安全**
- 写入前自动创建备份（保留最近10份）
- 配置文件与数据文件分离存储
- 敏感信息（API密钥）独立配置管理

### 6.1 竞品对比

| 维度 | PlanMosaic | 番茄Todo | Forest | 墨墨背单词 |

### 6.2 差异化优势

1. **日程+学习一体化**：市面上唯一将日程管理与词汇学习深度融合的产品
2. **AI深度集成**：不是简单的AI对话，而是通过工具调用实现日程的自动化管理
3. **深度规划**：提供SWOT、ROI等专业分析工具，超出一般日程管理产品的范畴
4. **完全免费**：用户只需自备API密钥，无任何功能限制
5. **多平台原生**：Android原生+Electron桌面+Uni-app跨平台，体验一致
